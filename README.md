Malicious Email Scorer

A secure client-server system designed to analyze and score incoming emails for phishing and security risks directly within the Gmail interface.

Tech Stack
Languages: Python, JavaScript (Google Apps Script)
Backend: FastAPI
Frontend: Gmail Workspace Add-on

Core Features
Real-time Analysis: Seamlessly integrates into Gmail to evaluate suspicious emails on the fly.
Threat Scoring: Utilizes a Python-based backend API to process email content and assign a concrete security risk score.
Client-Server Architecture: Demonstrates secure and efficient data transfer between a client-side Google Add-on and a standalone FastAPI backend.
